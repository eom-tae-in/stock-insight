## 구현 내용

### 1. .env.example 생성 (루트 디렉토리)

```bash
# SerpAPI (Google Trends 데이터)
# 무료 키 발급: https://serpapi.com/docs/google-trends-api
# 월 100회 무료 조회 제한
SERPAPI_KEY=your_free_api_key_here
```

### 2. .env.local (사용자 로컬, git 무시됨)

```bash
SERPAPI_KEY=actual_api_key_from_serpapi
```

### 3. src/lib/env.ts 수정 (기존 파일 확장)

```typescript
import { getDatabase } from './db'

// 환경 변수 로드
export const SERPAPI_KEY = process.env.SERPAPI_KEY
export const NODE_ENV = process.env.NODE_ENV || 'development'
export const DB_PATH = 'data/stock-insight.db'

// 환경 변수 검증
if (!SERPAPI_KEY && NODE_ENV === 'production') {
  throw new Error('SERPAPI_KEY is not defined in environment variables')
}

if (!SERPAPI_KEY) {
  console.warn(
    '⚠️ SERPAPI_KEY is not defined in .env.local\n' +
      'Google Trends data collection will fail.\n' +
      'Please add SERPAPI_KEY to .env.local and restart the development server.'
  )
}

// 앱 초기화 함수
export function initializeApp(): void {
  try {
    // 데이터베이스 초기화 (싱글톤, 첫 호출 시만 실행)
    getDatabase()
    console.log('✅ Application initialized successfully')
    logEnvironment()
  } catch (error) {
    console.error('❌ Failed to initialize application:', error)
    throw error
  }
}

// 환경 정보 로깅 (개발 단계용)
function logEnvironment(): void {
  console.log('🔧 Environment Configuration:')
  console.log(`  NODE_ENV: ${NODE_ENV}`)
  console.log(`  SERPAPI_KEY: ${SERPAPI_KEY ? '✅ Set' : '❌ Not set'}`)
  console.log(`  DB_PATH: ${DB_PATH}`)
}
```

## 환경 변수 정보

### SERPAPI_KEY (필수)

- **설명**: Google Trends 데이터를 수집하기 위한 SerpAPI API 키
- **발급 방법**: https://serpapi.com → Sign up → API Key 복사
- **제한**: 월 100회 무료 조회 (개인 개발자 충분)
- **사용처**: src/lib/api/trends.ts (F004 트렌드 수집)

### NODE_ENV (자동)

- **기본값**: 'development'
- **빌드 시**: 'production'

### DB_PATH (상수)

- **값**: 'data/stock-insight.db'
- **생성**: npm run dev 시 자동 생성
